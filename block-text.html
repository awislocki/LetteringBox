How it works: Each line is measured and scaled to fill the block width. Letters are spaced tightly (negative spacing) so they physically overlap. Lines are vertically overlapped so rows connect. The canvas bitmap is traced into a single welded vector path. Score lines (blue) mark letter boundaries for engraving. Cut outline (red) marks the outer edge for cutting.
