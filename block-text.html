How it works: Each line is stretched to fill the block width using SVG textLength. Lines are sized to fill the block height. The SVG text elements ARE the cut paths — import into laser software, select all, convert text to path (Path → Object to Path in Inkscape), then union (Path → Union). Result: one connected cuttable piece. Score lines mark letter boundaries for optional engraving.
